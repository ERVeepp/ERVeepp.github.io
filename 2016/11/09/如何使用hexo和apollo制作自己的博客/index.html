如何使用hexo和apollo制作自己的博客

Nov 9, 2016

一：node环境

建立一个空的文件夹，可以叫blog。
然后运行如下命令：
npm install –save
npm install -g hexo
npm install hexo-deployer-git –save

二：运行hexo

hexo-renderer-jade
hexo-generator-feed
hexo-generator-sitemap
hexo-browsersync
hexo-generator-archive

三：常用的一些操作

hexo init
hexo c
hexo g
hexo s
hexo d

apollo主题和Vue的官网有点像，所以……这里是主题地址: apollo
